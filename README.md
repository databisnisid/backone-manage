# BackOne Manage

## Installation
	virtualenv venv -p python3
	source venv/bin/activate
	pip install -r requirements.txt
	python manage.py migrate
	python manage createsuperuser

## Settings
	- Login as superuser
	- Create Controllter
	- Create Features
	- Create Organizations
	- Create user with Organizations
	
## Running
	- source /venv/bin/activate
	- ./start.ds
